   → flutter clean
   → flutter run

Problem: Map shows blank/gray
→ Solution: Check API key is added correctly to configs

Problem: Location not showing
→ Solution: Grant location permissions, enable GPS

Problem: Markers not visible
→ Solution: Check services have valid Kigali coordinates

Problem: App crashes on map screen
→ Solution: Verify API key is set (not placeholder text)
